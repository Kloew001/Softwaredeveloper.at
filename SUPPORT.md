# Support Policy

## Supported Versions

Only the latest published major/minor version is actively supported.

## What support includes

- Security fixes
- Critical bug fixes
- Documentation clarifications

## What support does not include

- Custom feature development for individual consumers
- Support for end-of-life framework/runtime versions

## Contact

For questions and support requests, open a GitHub issue.

For security issues, follow `SECURITY.md` and do not open a public issue.
